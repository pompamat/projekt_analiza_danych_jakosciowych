# Projekt Analiza Danych Jakościowych
Projekt ma na celu analizę danych dotyczących satysfakcji pasażerów Inve stico Airlines. Mając obserwacje klientów na temat podróży i zgłaszanego poziomu satysfakcji z poszczególnych aspektów lotu, staramy się dopasować model liniowy, które przewiduje, czy są oni zadowoleni z usługi, czy nie.
